# TruCare 25.2 — What's New

## Core

### Special Instructions Pop-up
TruCare now has a configurable pop-up displaying member Special Instructions whenever a user enters a member record (via search, task, etc.). New in 25.2.

### Manual Task Timezone Fix
Manual task due-date times now display in the user's local timezone (resolves CP-100005 from 24.2/25.1).

## Utilization Management

### UM Compliance Decision Table
A new UM Compliance decision table supports compliance rule management with a user-friendly interface. New in 25.2.

## Care Management

### Correspondence Rule — Historical Service Level
A new Historical Service Level attribute was introduced for the Program object in correspondence rules.
