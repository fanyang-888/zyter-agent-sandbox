# TruCare 25.1 — What's New

## Core

### Temporary Member Record Edit
Users can now edit temporary member records (First, Middle, Last name, DOB, Gender, SSN). New in 25.1 — not available in 24.2.

### Auto Save on System Timeout
Auto Save now protects open artifacts when the system logs a user out for inactivity. Work is restored on next login.

## Utilization Management

### InterQual 2025
TruCare 25.1 is certified with InterQual v2.1 and v2.2.

## Care Management

### Personal Medication List
A new Personal Medication List feature benefits care managers, pharmacists, and members with a centralized medication view. New in 25.1.

### Assessment Split-Screen
A split-screen lets users view clinical data and document assessments simultaneously. (Known issue for HCS members: CP-100006, still open.)
