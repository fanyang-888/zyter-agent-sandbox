# TruCare 24.2 — What's New

## Core

### Manual Tasks
Manual tasks can now set a due-date time. (Note: in 24.2 this time displays in UTC — see CP-100005, fixed in 25.2.)

### Case Closure Alert
A new alert dialog appears when a case manager clicks Submit on case closure, displaying outstanding items.

## Utilization Management

### MCG CareWebQI
TruCare 24.2 is certified with CareWebQI for concurrent review workflows.

## Care Management

### Care Plan Review Documentation
Users can document care plan reviews; the system records the last review date automatically.
